# Customer_Purchase_Behavior_Analysis
EDA on customer transactions using Python

## Tools
Python (Pandas, NumPy, Matplotlib, Seaborn), Google Colab

## Project Description
Performed exploratory data analysis on transactional customer data to understand purchase behavior, pricing patterns, and category-wise revenue contribution.

## Achievements/Results
- Identified high-value customers and top-performing categories
- Visualized order value distribution, repeat purchases, and pricing patterns

## Key Insight
Exposed repeat purchase patterns and pricing trends, highlighting opportunities to boost revenue and improve customer engagement.
